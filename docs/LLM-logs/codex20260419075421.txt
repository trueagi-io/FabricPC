  - Causal-only: diag_mean 0.6316, final_mean 0.3428, forget 0.2888
  - Causal+EWC: diag_mean 0.6356, final_mean 0.3183, forget 0.3173

  Interpretation:

  - EWC slightly improves diagonal task accuracy over causal-only: +0.0040
  - But it hurts the metric that matters most for continual retention:
      - final_row_mean: -0.0245 vs causal-only
      - forgetting: +0.0285 vs causal-only
  - It also increases summed task training time:
      - causal-only: 533.3s
      - causal+EWC: 584.9s

  So the current conclusion is: keep the causal tuning, do not enable EWC by default for this CIFAR setup.

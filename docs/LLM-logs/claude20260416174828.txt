 Problem Statement                                                                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                                                                  
 Promotions are counted but never applied. The SequentialTrainer._register_transweave_task_end() method:                                                                                                                                                                                                          
 1. Calls compute_demotion_transport() which returns promotion candidates                                                                                                                                                                                                                                         
 2. Counts the candidates (total_promotions += len(result.promotion_candidates))

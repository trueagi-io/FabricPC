  2. Root Cause Analysis: Why Forgetting Persists                                                                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                                                                                  
  The Fundamental Bottleneck                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                  
  Input → Columns (task-specific) → Aggregator (FULLY CONNECTED) → Output                                                                                                                                                                                                                                         
                                          ↑                                                                                                                                                                                                                                                                       
                              THIS IS THE PROBLEM                                                                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                                                                                  
  Even with perfect column isolation:                                                                                                                                                                                                                                                                             
  1. Aggregator connects ALL columns to ALL output classes                                                                                                                                                                                                                                                        
  2. Training new tasks changes aggregator weights                                                                                                                                                                                                                                                                
  3. Changed aggregator weights alter activations reaching frozen output neurons                                                                                                                                                                                                                                  
  4. Result: ~28-36% forgetting even with gradient protection                                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                                                                  
  Evidence from Logs                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                                  
  claude20260416191935.txt:                                                                                                                                                                                                                                                                                       
  "Even with frozen column outputs, aggregator weights change during new task                                                                                                                                                                                                                                     
  training, affecting inputs to frozen output neurons"                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                                                  
  "Fundamental architecture limitation... Need task-specific pathways -                                                                                                                                                                                                                                           
  partition aggregator neurons by task"

  This is the most important fix. The aggregator must partition its neurons:                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                  
  class PartitionedAggregator:                                                                                                                                                                                                                                                                                    
      """                                                                                                                                                                                                                                                                                                         
      Aggregator with task-specific pathways.                                                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                                                                  
      Architecture:                                                                                                                                                                                                                                                                                               
      - Input: (batch, num_columns, memory_dim)                                                                                                                                                                                                                                                                   
      - Each task owns aggregator_dim/num_tasks neurons                                                                                                                                                                                                                                                           
      - Task t's columns connect only to neurons [t*k : (t+1)*k]                                                                                                                                                                                                                                                  
      - Output: (batch, aggregator_dim) with only task-specific neurons active                                                                                                                                                                                                                                    
      """                                                                                                                                                                                                                                                                                                         
      def __init__(self, aggregator_dim, num_tasks, num_columns):                                                                                                                                                                                                                                                 
          self.neurons_per_task = aggregator_dim // num_tasks                                                                                                                                                                                                                                                     
          # Separate weight matrices per task                                                                                                                                                                                                                                                                     
          self.task_weights = [                                                                                                                                                                                                                                                                                   
              init_weights((num_columns * memory_dim, self.neurons_per_task))                                                                                                                                                                                                                                     
              for _ in range(num_tasks)                                                                                                                                                                                                                                                                           
          ]                                                                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                                                                  
      def forward(self, x, task_id):                                                                                                                                                                                                                                                                              
          # Only use this task's pathway                                                                                                                                                                                                                                                                          
          return x @ self.task_weights[task_id]

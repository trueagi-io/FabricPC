  Key Findings                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                  
  The attention mechanism works but isn't better for this task because:                                                                                                                                                                                                                                           
  1. Shared weights problem: Attention weights (Q, K, V) are global and update when training any task, causing forgetting                                                                                                                                                                                         
  2. Higher forgetting: 0.4095 vs 0.2863 - the attention weights learn to focus on recent tasks                                                                                                                                                                                                                   
  3. 6x slower: Complex attention computation vs simple linear projection                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                                  
  Recommendation                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                  
  The linear aggregator (default, no --attention) is better for Split-MNIST continual learning:                                                                                                                                                                                                                   
  - Higher accuracy                                                                                                                                                                                                                                                                                               
  - Lower forgetting                                                                                                                                                                                                                                                                                              
  - Faster training                                                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                                                  
  The attention approach would need task-specific attention heads (separate Q/K/V per task) or frozen weights for past tasks to match the linear aggregator's continual learning performance.
